Algorithm 
1. For Profile Update (updateProfile method): 
a. Validate updated information format and requirements 
b. Check for duplicate email conflicts in public_users table 
c. Begin database transaction 
d. Update user record with new information 
e. Commit transaction and return success status 
2. For Password Update (updatePassword method): 
a. Hash new password using bcrypt encryption 
b. Update password field in database 
c. Save updated record 
d. Return success confirmation 
3. For Password Validation (validateCurrentPassword method): 
a. Retrieve current password hash from database 
b. Compare provided password with stored hash 
c. Return boolean validation result
